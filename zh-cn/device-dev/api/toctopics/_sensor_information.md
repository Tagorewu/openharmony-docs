## **详细描述**<a name="section1136374581083932"></a>

定义传感器基本信息。

一个传感器设备信息包括传感器名字、设备厂商、固件版本号、硬件版本号、传感器类型编号、传感器标识、最大量程、精度、功耗。

**Since：**

2.2
