# 键值

KeyCode模块提供了按键类设备的键值。

> ![icon-note.gif](public_sys-resources/icon-note.gif) **说明：**
> 本模块首批接口从API version 9开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

## 导入模块

```js
import {KeyCode} from '@ohos.multimodalInput.keyCode'
```
