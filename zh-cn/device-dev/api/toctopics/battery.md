## **详细描述**<a name="section1680766641083931"></a>

提供获取、订阅电池信息的接口。

电池模块为电池服务提供的获取、订阅电池信息的接口。 服务获取此模块的对象或代理后，可以调用相关的接口获取电池信息、订阅电池信息的变化。

**Since：**

3.1

**Version：**

1.0
