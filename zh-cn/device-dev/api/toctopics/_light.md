## **详细描述**<a name="section1385532827083931"></a>

灯模块对灯服务提供通用的接口能力。

灯模块为灯服务提供通用的接口去访问灯驱动。 服务获取灯驱动对象或代理后，可以调用相关的APIs接口获取灯信息、打开或关闭灯，并根据灯ID设置灯闪烁模式。

**Since：**

3.1
