# Running


## Viewing Execution Result

After the sample code is compiled, burnt, run, and debugged, restart the development board. If the following messages are displayed, the image is running correctly:

  
```
ready to OS start
FileSystem mount ok.
wifi init success!
[DEMO] Hello world.
```
